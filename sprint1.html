Vandaag heb ik mijn eigen html gemaakt voor mijn visite kaartje. Vandaag beantwoord ik ook de drie studievragen hieronder voor mezelf.

  • Wat is het nut van een breakdown schets?
  • Waar moet je op letten als je een breakdown schets maakt?
  • Hoe zorg ik ervoor dat ik mijn tekst/plaatjes op de juiste plek krijg?

Het nut van een breakdownschets is dat je voor jezelf al de code kan schrijven. Je schrijft tijdens het schetsen als de code op en hierdoor kom je er ook achter hoe je het eruit wilt laten zien. Verder wil ik uitzoeken hoe CSS werkt en waar elke code voor staat zodat ik precies weet hoe ik mijn plaatjes en tekst naar de juiste plek krijg.
